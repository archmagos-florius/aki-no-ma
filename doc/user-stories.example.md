# Example: FocusTasks
```markdown
## UC-1: Sign up
- As a new user, I want to create an account so my tasks are saved.

## UC-2: Create a task
- As a user, I want to create a task with title, optional description, due date, and project.

## UC-3: View tasks by project
- As a user, I want to see all tasks in a selected project.

## UC-4: Mark task as done
- As a user, I want to mark a task as done so it no longer appears in my active list.

## UC-5: Start a focus session
- As a user, I want to start a focus session on selected tasks so I can concentrate on them for a set time.

## UC-6: Add reminder to a task
- As a user, I want to set a reminder for a task so I get notified at a specific time.

## UC-7: See overdue tasks
- As a user, I want to see tasks that are overdue so I can prioritize them.

## UC-8: Tag tasks
- As a user, I want to add tags to tasks and filter by tag.
```

--

# Template
```markdown
# Your extras
## UC-<N>: <Short name>
- As a <type of user>, I want <action> so that <benefit>.
```
