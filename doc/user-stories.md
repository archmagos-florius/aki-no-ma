# Food Ordering App (Aki'noma)

## Story numberal designations

### UC-0**: regular user stories
- these are the stories for regular users either  logged in or not

### UC-1**: admin user stories
- these are stories with executive access to the application with the power to manage all other aspects including orders/menus/etc.

### UC-2**: driver user stories
- these are stories for users with the driver role they have a subset of admin powers relating to order delivery

## UC-001: Sign up
- As a new user, I want create an account so i can track orders, enter raffles, update orders

## UC-002: Create Carts
- As a user, i want to add items to my cart so that they can persist for a certain amount of time at least in case i can't make up my mind right away

## UC-003: Create Orders
- As a user, i want to take the cart and all the attached information add contact information and create an order so i can receive my Food

## UC-004: Track Order
- As a user, i want to be able to track the status of my order to make sure that my order is being taken care of (I will also receive notfiications for these updates)

## UC-005: Update Order
- As a user, i want to be able to edit the order the i have placed so i don't receive the wrong items

## UC-006: Checkout cart
- As a user i want to take my cart and enter the checkout process so that i can create an order using my current cart
